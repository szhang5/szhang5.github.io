1818. Minimum Absolute Sum Difference

Medium Problem

April 15, 2021 Shiyun Zhang

Problem Analysis

You are given two positive integer arrays nums1 and nums2, both of length n.
The absolute sum difference of arrays nums1 and nums2 is defined as the sum of |nums1[i] - nums2[i]| for each 0 <= i < n (0-indexed).
You can replace at most one element of nums1 with any other element in nums1 to minimize the absolute sum difference.
Return the minimum absolute sum difference after replacing at most one element in the array nums1. Since the answer may be large, return it modulo 10**9 + 7.
To minimize the absolute sum difference, we only need to find the maximum difference pair (x, y) x in nums1, y in nums2 and then find the minimum difference element for x in nums1

For example:
nums1 = [1, 10, 4, 4, 2, 7]
nums2 = [9. 3, 5, 1, 7, 4]
The difference are 8, 7, 1, 3, 5, 3 --> sum = 8+7+1+3+5+3 = 27
So the maximum difference pair is (1, 9)
Now find an element in nums1 to replace 1 to minimize the sum difference:
|10-9| = 1, |4-9| = 5, |4-9|=5, |2-9| = 7, |7-9|=2
So 10 will be the best element to replace 1
Final result should be 27 - 8 + 1 = 20
