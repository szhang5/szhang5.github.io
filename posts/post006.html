1792. Maximum Average Pass Ratio

Medium Problem

March 27, 2021 Shiyun Zhang

Problem Analysis

We need to calculate the maximum average pass ratio in this problem. We are given a 2D array that stands for classes which classi has passi students pass the exam and totali number of total students in the class.

To calculate the average pass ratio of the school we should:

  1. Calculate the average pass ratio of the class => classPassRatio = pass / total
  2. Calculate the average pass ratio of the school => schoolPassRatio = sum of classPassRatio / numberOfClasses

From the above steps, in order to get the maximum average pass ratio, we need to get the maximum sum of classPassRatio, which means we need to maximize the pass ratio when we assign extra students to the specific class. To do this we need to calculate the impact when we assign a extra student:

  • currentRatio = pass / total
  • afterAssignedExtraStudentRatio = pass+1 / total+1
  • Impact = afterAssignedExtraStudentRatio - currentRatio

To always get the largest impact, we will use the max heap data structure. We know that in python, heapq can only generate min heap for us but we can use negative number to turn the min heap to max heap
