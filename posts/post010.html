1814. Count Nice Pairs in an Array

Medium Problem

April 15, 2021 Shiyun Zhang

Problem Analysis

    A pair of indices (i, j) is nice if it satisfies all of the following conditions:
    1) 0 <= i < j < nums.length
    2) nums[i] + rev(nums[j]) == nums[j] + rev(nums[i])
    According to condition2, we have nums[i]-rev(nums[i]) == nums[j]-rev(nums[j])
    When counting the nice pairs in an array, we can first calculate how many nums[i]-rev(nums[i]) have the same value.
    For example, nums = [13, 10, 35, 24, 76]
    13 - rev(13) = 13 - 31 = -22
    10 - rev(10) = 10 - 1 = 9
    35 - rev(35) = 35 - 53 = -22
    24 - rev(24) = 24 - 42 = -22
    76 - rev(76) = 76 - 67 = 9
    Therefore, we have counter = {-22 : 3, 9 : 2}
    The combinations are as the following:
    1) 13 - 35 --> 13 - rev(13) == 35 - rev(35)
    2) 13 - 24 --> 13 - rev(13) == 24 - rev(24)
    3) 24 - 35 --> 24 - rev(24) == 35 - rev(35)
    4) 10 - 76 --> 10 - rev(10) == 76 - rev(76)

There are 2 same values of 9 so there are 2 * (2-1) // 2 = 1 combination
There are 3 same values of -22 so there are 3 * (3-1) // 2 = 3 combinations.
If there are 4 same values of X, there will be 4 * (4-1) // 2 = 6 combinations
E.g. [1,2,3,4] --> 1-2, 1-3, 1-4, 2-3, 2-4, 3-4
