343. Integer Break

Medium Problem

March 27, 2021 Shiyun Zhang

Problem Analysis

DP solution

Given the following examples:
2: 1+1 => 1 * 1 = 1
3: 2+1 => 2 * 1 = 2
4: 2+2 => 2 * 2 = 4
5: 3+2 => 3 * 2 = 6
6: 1+5 / 2+4 / 3+3 => 1*5 / 2*4/ 3*3 => 9
7: 1+6 / 2+5 / 3+4 => 1*9 / 2*6 / 3*4 => 12
8: 1+7 / 2+6 / 3+5 / 4+4 => 1*12 / 2*9 / 3*6 / 4*4 => 18

We can retrieve the corresponding maximum product from a specific number like 8 = 2 + 6 => 2 * 9 = 18. The maximum product of 6 is 9

Mathematical solution

Given the following examples:
7 = 3 + 4 = 12
8 = 3 + 3 + 2 = 18
9 = 3 + 3 + 3 = 27
10 = 3 + 3 + 4 = 36
11 = 3 + 3 + 3 + 2 = 54
12 = 3 + 3 + 3 + 3 = 81

3 is a magical number
