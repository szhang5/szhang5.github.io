1823. Find the Winner of the Circular Game

Medium Problem

April 15, 2021 Shiyun Zhang

Problem Analysis

Solution 1:
When n = 5, k = 2
i = 0
i = (0 + k - 1) % len(Arr) = 0 + 2 -1 = 1%5 = 1
Arr = [1,2,3,4,5] --> remove arr[1] --> Arr = [1,3,4,5]
i = 1
i = (1 + k -1) % len(Arr) = 1 + 2 - 1 = 2 % 4 = 2
Arr = [1,3,4,5] --> remove arr[2] --> Arr = [1,3,5]
i = 2
i = (2 + k - 1) % len(Arr) = 2 + 2 - 1 = 3 % 3 = 0
Arr = [1,3,5] --> remove arr[0] --> Arr = [3,5]
i = 0
i = (0 + k - 1) % len(Arr) = 0 + 2 - 1 = 1 % 2 = 1
Arr = [3,5] --> remove arr[1] --> Arr = [3]
Result: 3

Solution 2 (Easier understanding)
When n = 5, k = 2
[1,2,3,4,5] remove 2
--> start counting from 3: [3,4,5,1] remove 4
--> start counting from 5: [5,1,3] remove 1
--> start counting from 3: [3, 5] remove 5
[3] --> result 3
