1789. Maximum Number of Consecutive Values You Can Make

Medium Problem

March 26, 2021 Shiyun Zhang

Problem Analysis

We have a list of coins and we need to return the maximum number of consecutive integer values that we can make with the coins starting from and including 0.

Let’s compare two examples:

Example 1:
Coins = [1, 1, 1, 4]
0 : 
1: 1
2: 1+1
3: 1+1+1
4: 4
5: 4+1 (we can do this because we have 4 and 1)
6: 4+1+1 (we can do this because we have 4 and 2)
7: 4+1+1+1 ( we can do this because we have 4 and 3)
8?? (we have 6 and 2 why can not add it up to 8? because the maximum sum is 7 based on the coins)
Example 2:
Coins = [1, 4, 6]
0: 
1: 1
2?? (we have 1 why cannot use 1+1? because we only have one ‘1’ in the coins)
