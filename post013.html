1793. Maximum Score of a Good Subarray

Hard Problem

April 15, 2021 Shiyun Zhang

Problem Analysis

You are given an array of integers nums (0-indexed) and an integer k.

The score of a subarray (i, j) is defined as min(nums[i], nums[i+1], ..., nums[j]) * (j - i + 1). A good subarray is a subarray where i <= k <= j.

Return the maximum possible score of a good subarray.

According to the problem, the score is calculated by:
(minimum number in an subarray) * (length of the subarray)

For example: nums = [1,4,3,7,4,5], k = 3
Then we need to see the kth element as center of the subarray

1) Subarray = [3] --> score = 3 * 1 = 3
2) Subarray = [4,3] --> score = 3 * 2 = 6
3) Subarray = [3, 7] --> score = 3 * 2 = 6
4) Subarray = [4,3,7] --> score = 3 * 3 = 9

From the above we can see, when the minimum number in the subarray does not change, the maximum score will be determined by the length. We should find the longest length for the specific minimum number. E.g. when minimum number is 3, we should get subarray = [4,3,7,4,5] which score is 3 * 5 = 15 and then switch to minimum number = 1, where subarray = [1,4,3,7,4,5] and score is 1 * 6 = 6.
